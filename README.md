# Plainview

Initial scaffold for the Plainview platform.

- API: `services/api` (Fastify, TypeScript)
- Dashboard: `apps/dashboard` (Vite + React + TS)
- Shared types: `packages/shared`

Quickstart
- Install deps: `npm install`
- API: `npm run dev -w @plainview/api`
- Dashboard: `npm run dev -w @plainview/dashboard`
