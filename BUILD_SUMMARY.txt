================================================================================
API ENDPOINTS
================================================================================

Core:
  GET  /health           → System status & uptime
  GET  /                 → API info
  GET  /modules          → Domain modules list

ValveOps:
  GET  /valves           → List valves with torque
  POST /valves/:id/actuate → Simulate rotation (1.2s)

PipelineGuard:
  GET  /pipeline/alerts  → Leak incidents

RigSight:
  GET  /rig/cameras      → Camera list

FlowIQ:
  GET  /flow/health      → Component health scores

Real-Time:
  GET  /events           → SSE stream (5s heartbeat + events)

Event Types: 8
  - valve.actuation.requested/completed
  - leak.alert
  - telemetry.tick
  - anomaly.detected
  - mission.started/completed
  - device.status
